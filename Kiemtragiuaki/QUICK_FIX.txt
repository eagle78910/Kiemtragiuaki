📝 NOTES
═══════════════════════════════════════════════════════════════════════════════

The checkbox issue was a common ASP.NET problem:
- Unchecked checkboxes don't send any value in HTML
- This caused ASP.NET binding to fail silently
- The fix is to add a hidden input with the unchecked value

This is the standard pattern in professional ASP.NET applications.
